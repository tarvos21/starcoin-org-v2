    版本记录

    Starcoin 版本记录

    v0.3.1

    引入Epoch 以及叔块机制。

    v0.3.1

    稳定化基本类型以及 Stdlib。

    v0.4.2

    Account 和 State 命令增强.

    v0.4.3

    同步 Bug 修复。

    v0.4.4

    Stdlib 增加 Spec 形式化验证，覆盖率达到80%.

    v0.4.5

    重构叔块、Open block，create_block_template模块以及服务注册器实现。

    v0.4.6

    支持用户自定义Chain, 支持多个Stdlib版本，实现了 Token 投票治理机制

    v0.5.0

    Service 重构以及一系列不兼容变更

    v0.5.1

    重构共识配置，在genesis初始化时增加了预挖、时间锁功能

    v0.7.0

    重构共识，实现cryptonight的新算法，增强和优化累加器，动态调整 block gas limit

    v0.7.1

    一些清理和 bug 修复

    v0.8.0

    同步优化以及 stdlib 重构

    v0.9.0

    Stdlib v7 版以及 p2p 网络优化

    v0.9.1

    重构miner、BlockTemplate、OpenedBlock, 同步数据到搜索引擎

    v0.9.2

    优化网络以及同步服务，提高同步速度，增强同步容错机制

    v0.9.3

    Miner 客户端实现断网自动重连机制以及 bug 修复

    v0.9.4

    增加同步 peer 打分机制

    v0.10.0

    重构 StateDB 和 AccessPath

    v0.10.1

    重构 BlockHeader 以及Module 兼容性检查, 优化交易广播

    v1.0.0-alpha

    实现 list-resource API并优化 rpc, 系统加固

    v1.0.0-alpha.1

    将 move-vm 升级到最新版本以支持新的 Move 语言功能

    v1.0.0-beta

    升级 MoveVm 以及重构 Stdlib. 支持新的 abilities 语法，支持 Module 中定义的 ScriptFunction，准备正式启动新的 Barnard 网络。

    v1.0.0-beta.1

    生成 Barnard 网络的创世块并包含到二进制中，优化区块广播策略.

    v1.0.0-beta.2

    优化同步以及网络消息处理，新增两个配置项，方便节点配置自己支持的最大节点数。

    v1.0.0-beta.3

    优化同步时候的 peer 选择策略，保证同步更及时。

    v1.0.0-beta.4

    网络协议更新，为未来新增协议或者支持轻节点做准备。

    v1.0.0-beta.5

    网络协议升级，优化交易广播机制；支持 stratum 矿池协议；合约升级方式优化：支持通过治理方式实现不兼容性升级，以及实现通过链上治理进行硬分叉(sip-4).

    注意: 节点请尽快升级到最新版本，即将进行一次硬分叉测试，会导致旧版本的节点无法执行最新区块。

    v1.0.0-beta.6
    改进 stdlib 升级 <！–更多–> [stratum] 修复 stratum bug。 [stdlib] 批量转账v2 (#2463) [k8s] 更新k8s配置 (#2459) [cmd] 增加一个 cmd 来提交升级 vm 配置的提案 (#2466) [transaction-builder-generator] 修复 python 反序列化 bug (#2467) [stdlib] 修复 stdlib 升级的错误 (#2461) [stdlib] 使 stdlib 升级不受 PublishOption的影响，改进错误处理，并修复错误 (#2450) [stdlib] 实现 collection2 (#2453) [events] 生成用于 SDK 的事件数据格式(#2458) [network-p2p] 增加 max_connections_per_address 配置，限制单 ip 链接数 (#2454) [ci] 当某些平台失败时继续发布 (#2448)
    v1.0.0-rc

    引入国库机制，调整经济模型，为主网发布做准备

    v1.0.0

    生成创世块，主网启动。

    v1.1.0

    Account 以及 Miner 相关的接口和命令增强

    v1.2.0

    发布 Move CLI 工具以及重构账号的 authentication key 校验机制，简化账号的初始化方式。

    注：本版本包含硬分叉特性，将在 stdlib 升级到 v5 版本后激活。请升级到最新版本，等待升级提案发起后，进行链上投票。关于投票的详细说明会在升级提案提交后给出。
